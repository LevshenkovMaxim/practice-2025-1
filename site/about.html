О проекте

Партнёр проекта

Нашим партнёром выступает компания Аарон авто. Это дилерский центр, занимающийся продажей и обслуживанием автомобилей разных марок. Основная деятельность компании - кузовной ремонт. Сейчас, чтобы записаться на обслуживание, клиентам приходится обращаться к мастерам-консультантам очно или по телефону. Наш проект призван сделать этот процесс удобнее как для клиентов, так и для сотрудников автосалона.

Цель проекта

Основной целью проекта является создание "электронного мастера-консультанта". Это представленное на различных платформах программное обеспечение для записи клиентов дилерского центра на техобслуживание автомобиля. В нашем случае, проект будет представлен на стоечных терминалах в здании организации, в виде веб-приложения на сайте партнёра и в качестве мобильного приложения на Android и iOS. Наш проект актуален для партнёра: наличие такого электронного сервиса не только обеспечивает удобство, но и повышает конкурентоспособность компании.

Основные задачи проекта:

  • Сбор информации о конкурентах

    Провести анализ рынка и изучить похожие электронные сервисы конкурентов. Выявить сильные и слабые стороны. Использовать полученные данные при создании собственных систем.

  • Изучение доступных платформ

    Обозначить план разработки программного обеспечения, платформы. Определиться с языками программирования, на которых будет написано ПО.

  • Консультация с партнёром

    Проконсультироваться с организацией-партнёром. Обсудить выбор платформ, способ интеграции в инфраструктуру и БД организации. Выбрать подходящие стоечные терминалы для размещении непосредственно в автосалоне.
